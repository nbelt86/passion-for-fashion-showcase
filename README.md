# Passion For Fashion

Team project completed during Le Wagon Web Development Bootcamp.

## Project Overview

Passion For Fashion is an AI-powered fashion recommendation platform built during the Le Wagon Web Development Bootcamp.

Users complete a style quiz and upload a photo. The application generates personalized fashion recommendations based on lifestyle, preferences, and uploaded images.
